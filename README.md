# JARVIS V6 – Memory System

JARVIS V6 introduces a persistent memory system that allows the AI to store and recall information across sessions.

## 🚀 Features

* 🧠 Persistent memory storage
* 🔁 Context awareness
* 📂 Data recall system
* ⚡ Smarter responses

## 🛠️ Tech Stack

* Electron
* AI APIs
* File-based storage

## 🎯 Goal

To build an AI that remembers, learns, and evolves over time.

## 🔥 Future Plans

* Long-term memory system
* Personalization
* Smarter recall logic
* Cross-session learning
